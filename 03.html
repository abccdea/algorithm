函数增长

插入排序运行时间：$\Theta(n^2)$

归并排序运行时间：$\Theta(n \lg n)$

算法的渐近效率：输入规模足够大时，运行时间与增长量级有关

渐近记号

插入排序最坏情况的运行时间 $an^2+bn+c$，写成 $\Theta(n^2)$ 除去了该函数的细节。同样的这种方式可以用来描述空间，还可以描述任何输入的运行时间，不仅是最坏情况。

$\Theta$ 记号：给出函数的上界和下界

$\Theta(g(n))=\{f(n): 存在正常量c_1、c_2和n_0，使得对所有n \geq n_0，有 0 \leq c_1g(n) \leq f(n) \leq c_2g(n)\}$

$f(n)\in\Theta(g(n))$也可以写为 $f(n)=\Theta(g(n))$

渐近紧确界：对所有 $n \geq n_0$，$f(n)$ 在一个常量因子内等于 $g(n)$

$O$ 记号：给出函数的上界

$O(g(n))=\{f(n): 存在正常量c和n_0，使得对所有n \geq n_0，有 0 \leq f(n) \leq cg(n)\}$

$\Omega$ 记号：给出函数的下界

$\Omega(g(n))=\{f(n): 存在正常量c和n_0，使得对所有n \geq n_0，有 0 \leq cg(n) \leq f(n) \}$

定理 $f(n)=\Theta(g(n))$，当且仅当 $f(n)=O(g(n))$ 且 $f(n)=\Omega(g(n))$

$\omicron$ 记号：给出函数的上界，没有等号

$\omicron(g(n))=\{f(n): 存在正常量c和n_0，使得对所有n \geq n_0，有 0 \leq f(n) < cg(n)\}$

$ \displaystyle \lim_{ n\to\infty } \frac {f(n)} {g(n)} = 0$

$\omega$ 记号：给出函数的下界，没有等号

$\Omega(g(n))=\{f(n): 存在正常量c和n_0，使得对所有n \geq n_0，有 0 \leq cg(n) < f(n) \}$

$ \displaystyle \lim_{ n\to\infty } \frac {f(n)} {g(n)} = \infty$

函数和实数比较

对于任意两个实数$a$ 和 $b$，三种情况必有一个成立：$a < b$、$a=b$ 和 $a> b$。

但是函数却不成立

例如：

$n$ 和 $n^{1 + \sin n}$，幂值在0和2之间摆动，介于两者之间
